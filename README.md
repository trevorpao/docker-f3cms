# docker-simple-lamp  
```  
 conf   -----   config  
    apache  -   apache vhost  
  
 log    -----   log  
    apache  -   apache log  
    mysql   -   mysql log  
    php     -   php log  
   
 www    -----   website  
 ```  
  
 ##start the container  
 `./up.sh`  
  
 ##close the container  
 `./down.sh`
